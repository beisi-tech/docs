# HTML 教程

HTML（HyperText Markup Language，超文本标记语言）是构建网页的标准标记语言。它使用标签来描述网页的结构和内容，是 Web 开发的基础。

## HTML 的特点

- **标记语言**：使用标签来描述内容结构
- **平台无关**：可以在任何操作系统和浏览器中运行
- **易于学习**：语法简单，标签直观
- **标准化**：遵循 W3C 标准
- **可扩展**：支持自定义标签和属性
