# JavaScript 教程

JavaScript 是一种高级的、解释型的编程语言，主要用于 Web 开发。它是 Web 三大核心技术之一（HTML、CSS、JavaScript），也是世界上最流行的编程语言之一。


## JavaScript 的特点

- **动态类型**：变量类型在运行时确定
- **解释型语言**：无需编译，直接运行
- **跨平台**：支持多种操作系统和浏览器
- **事件驱动**：基于事件和回调的编程模式
- **函数式编程**：支持函数作为一等公民
